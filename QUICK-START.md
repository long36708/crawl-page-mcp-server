```bash

./start-optimized.sh

# 或直接启动
npm start
```

### 2. 基本用法
```json
{
  "name": "crawl_page",
  "arguments": {
    "url": "https://example.com",
    "format": "markdown",
    "timeout": 4000,
    "useCache": true
  }
}
```

### 3. 高性能配置
```json
{
  "url": "https://example.com",
  "format": "text",          // 最快格式
  "selector": "main",        // 精确选择
  "timeout": 3000,           // 短超时
  "useCache": true           // 启用缓存
}
```

## 🛠️ 故障排除

如果仍遇到超时：
1. 检查网络连接
2. 使用更短的timeout (2000-3000ms)
3. 启用缓存 `"useCache": true`
4. 使用CSS选择器减少处理量
